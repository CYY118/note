### 1.存放文章的数据库设计

##### 表名：bl_Article

| 昵称             | 类型    | 长度       |
| :--------------- | :------ | :--------- |
| id               | int     | 4          |
| bl_ArticleID     | varchar | 20         |
| bl_title         | varchar | 30         |
| bl_author        | varchar | 10         |
| bl_time          | date    |            |
| bl_show          | int     | 1(1是true) |
| bl_numCharacters | int     | 5          |
| bl_type          | varchar | 10         |
| bl_context       | text    |            |
```sql
create table bl_Article(
id int(4) primary key auto_increment,
bl_ArticleID varchar(20),
bl_title varchar(30),
bl_author varchar(10),
bl_time date,
bl_show int(1) check (bl_show in('1','0')) default '1',
bl_numCharacters int(5),
bl_context text
)
```
